Latest Projects

Geelong Football Club - Rotations Web Application

ASP.NET core MVC Application to visualize interchange data for the AFL team Geelong Cats.

New

For this project we have created a web-based application that provides the Geelong Football Club with a platform to maintain and monitor player interchange and rotations. The application utilises data collected from champion data regarding interchange times and provides a graphical representation of this data in multiple formats. This allows coaching staff, trainers, and management to identify optimal rotation strategies. Information regarding total substitutions, and blood rule or concussion substitutions are tracked to aid the club remain in line with substitution rules stipulated by the AFL.
