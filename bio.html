About Me

My experience in ICT is solid as I have been involved in the industry for over six years, working in a variety of support and system administration roles. However, since going back to school I have committed to a somewhat lateral career shift into programming and software development. For several reasons I have found the developer environment to best suit my personal strengths and interests.


The majority of my programming knowledge has come from study and projects undertaken during my degree over the past 3 years. What should be observed is I have plenty of hours using .Net & C#. I am very comfortable using cloud platforms to deploy applications and create/maintain virtual environments/servers. I was lucky enough to get the capstone project with the Geelong Football Club, which involves creating a web application to visualize interchange data. This has been delivered through a .Net Core MVC web site with a SQL server, hosted together on azure.


My career has had a lot of crossover and exposure to things such as, web development and content management systems, windows environment scripting such as PowerShell and batch files, command line and various consoles, dev-ops style work within the company and to third party software providers, following Agile methodology and principles, and project management, to name a few of the most obvious. I also have experience as administrator of database servers, including Microsoft SQL server 2008 & 2012.


The specific trigger for the career change came about once I started a Bachelor of Cyber Security majoring in cloud computing at Deakin University in 2016. In attempt to increase knowledge in my current sphere I accidentally found a new passion for code and software development. I am now in my final trimester having since shifted to a Bachelor of IT; with Programming and Cloud Computing as a double major. If all goes well I should finish with High Distinction in October.
